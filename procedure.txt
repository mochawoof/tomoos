TomoOS is based on Debian standard.

After installation:
su
usermod -aG sudo username

If you're on Virtualbox:
sudo adduser username vboxsf

Install Wayland:
sudo apt install weston xwayland waycheck

Install fun stuff (Firefox/media players):
sudo apt install firefox-esr mpv

Get disk usage:
df -H

To build without makefile:
gcc -c test.cpp
gcc test.o -o test.out -lX11

To clean:
rm *.o *.out

Install clock updater:
sudo apt install chrony
