TomoOS is based on Debian standard.

After installation:
su
usermod -aG sudo username

If you're on Virtualbox:
sudo adduser username vboxsf

Installing xorg:
sudo apt update &&
sudo apt install xorg xinit xterm

Install libx11 & gcc:
sudo apt install libx11-dev build-essential

Install firefox:
sudo apt install firefox-esr

Get disk usage:
df -H

To build without makefile:
gcc -c test.cpp
gcc test.o -o test.out -lX11

To clean:
rm *.o *.out

Install clock updater:
sudo apt install chrony
